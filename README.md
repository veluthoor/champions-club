# Champions Club

Monthly awards celebrating top All Access performers at Game Theory.

## February 2026 Awards

View the live page: [index.html](index.html)

### Award Categories

1. **Consistency King/Queen** - Most check-ins
2. **Multi-Sport Warrior** - Most sports explored
3. **Ultimate Variety Seeker** - Most activity combinations
4. **The Adventurer** - Most diverse activities
5. **Sport Specialist** - Most dedicated to single sport
6. **Facility Hopper** - Most facilities visited

### Features

- Game Theory brand colors (dark theme with orange accents)
- MongoDB-powered data from verified check-ins
- Responsive design with hover effects
- Activity tracking based on sport + activity type combinations

## Deployment

This page can be hosted on GitHub Pages or any static hosting service.

To enable GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages
3. Select source: Deploy from branch
4. Choose branch: main
5. Save

Your site will be available at: `https://[username].github.io/[repo-name]/`
